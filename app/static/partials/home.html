Twitter Health Surveillance

A Service to Monitor Diseases by Using Twitter.

Categorize tweets
about possible diseases.

Real-time data recollection means many disease tweets are saved in a day. Users can help categorize tweets by disease in an easy and quick matter using the dashboard.

Monitor outbreaks
in your area.

Users can monitor different disease outbreaks by filtering the data by area. They can also filter the data by an specific disease to monitor it around the globe.

View the latest statistics
on each outbreak on the go.

Users can access the latest statistics by reaching the website from any mobile device. Disease data is recorded in real-time which means that the site will always provide the latest results.
